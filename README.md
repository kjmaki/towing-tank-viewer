# Wave Basin Field Viewer

Interactive browser tools for visualising wave fields in a directional wave basin, based on the Dalrymple (1989) sidewall-reflection model.

**Live app:** https://kjmaki.github.io/towing-tank-viewer/

---

## Tank specifications

| Parameter | Value |
|---|---|
| Length | 100 m |
| Width | 6.6 m |
| Water depth *h* | 3 m |
| Paddles | 12 (full-span flap, *a* = *b*) |
| Half-width *b* | 3.3 m |

---

## Applet 1 — Regular Waves

Monochromatic oblique wave field generated by a snake-motion wavemaker.  
The field is the Dalrymple (1989) modal sum including both side-wall reflections, reconstructed from the even (cosine) and odd (sine) transverse modes.

### Controls

| Control | Definition |
|---|---|
| **θ** — direction | Paddle obliquity angle from tank axis (°). Larger θ steepens the crest angle across the width. |
| **T** — period | Wave period (s). Determines wavelength λ and wavenumber *k* via the dispersion relation ω² = *gk* tanh(*kh*). |
| **View length** | Down-tank window (m) shown in the plot. |
| **Contour bands** | Toggles smooth vs. banded (isoline) colour rendering. |
| **Designer** | Phase-aligns all propagating modes so the crest is clean at the test station *x*ₘ, matching how a wavemaker controller would be tuned experimentally. |
| **xₘ** | Target test station for the designer mode (m from paddle). |

### Key quantities displayed

| Quantity | Definition |
|---|---|
| λ | Wavelength = 2π/*k* (m) |
| *k* | Wavenumber, root of ω² = *gk* tanh(*kh*) (rad/m) |
| *kh* | Dimensionless depth. *kh* ≫ 1 → deep water; *kh* ≪ 1 → shallow water. |
| Propagating modes | Number of transverse modes with real *x*-wavenumber. Evanescent modes (imaginary *kx*) decay from the paddle and do not reach the test region. |

## Applet 2 — Irregular Waves

Directional irregular sea state realised in the tank: a JONSWAP frequency spectrum combined with a cos²ˢ directional spreading function, each (ω, θ) component reproduced as the Dalrymple modal field with a random phase, then superposed.

### Wave spectrum parameters

| Parameter | Definition |
|---|---|
| **Hs** | Significant wave height — mean of the highest third of waves (cm). Sets the overall energy scale. |
| **Tp** | Peak period (s) — period at the spectral peak *f*ₚ = 1/*Tp*. |
| **γ** | JONSWAP peakedness factor. γ = 1 → Pierson-Moskowitz (fully developed); γ = 3.3 → typical fetch-limited swell; γ = 7 → very narrow-banded. |
| **σ** — spread | Directional half-width (°) of the cos²ˢ spreading function. Larger σ → more spread, shorter crests. |
| **θ₀** — mean heading | Mean wave direction relative to tank axis (°). |

### Tank physics and limits

| Quantity | Definition |
|---|---|
| **θ_c** — cutoff angle | Critical angle asin(π / 2*bk*ₚ). Any component within ±θ_c of head-on cannot excite a transverse mode and radiates as an essentially plane wave. |
| Resolved energy | Fraction of directional energy at headings beyond ±θ_c — these components are genuinely short-crested in the tank. Shown as a percentage in the readout strip. |
| **Directional spread D(θ)** chart | Green shading = resolved (short-crested); red shading = below cutoff (collapses to plane wave); yellow line = mean heading θ₀. |
